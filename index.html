Welcome to Jon's raspbian repository

This site hosts get-iplayer packages for the Raspberry Pi (specifically, for Raspbian jessie), including backports of tools it requires:

  • get-iplayer
  • AtomicParsley
  • ffmpeg
  • rtmpdump

Recent Changes

Feb 2017: Wheezy Packages Retired

Wheezy is no longer supported - please update your apt sources to point to jessie instead, or simply (re)run the install script below.

Jan 2017: Expired Repository Signing Key

My GPG key expired in January 2017 - if you installed the repository signing key prior to this you will need to update the key, or simply (re)run the install script below.

Using this repository

To install the repository, get-iplayer and its dependencies, paste this line into a terminal window:

wget http://packages.hedgerows.org.uk/scripts/install.sh -O - | sh

To remove get-iplayer and the repository, paste this line into a terminal window:

wget http://packages.hedgerows.org.uk/scripts/uninstall.sh -O - | sh
The script uses sudo for some tasks, which will ask for your password.

Both scripts use sudo for some tasks, which will ask for your password. If you want to know in advance what the script will do, see install.sh or uninstall.sh.
