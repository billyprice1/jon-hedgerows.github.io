Welcome to Jon's raspbian repository

This site hosts software for the Raspberry Pi (specifically, for Raspian wheezy and Raspbian jessie), including

  • get-iplayer
  • AtomicParsley
  • rtmpdump

Expired Repository Signing Key

My GPG key expired in January 2017 - if you installed the repository signing key prior to this you will need to reinstall it:

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 0F5BFDFE

Using this repository

  1. Add the repository:

    Paste these lines into a terminal window -

    echo "deb http://packages.hedgerows.org.uk/raspbian wheezy/" | sudo tee
/etc/apt/sources.list.d/packages.hedgerows.org.uk.list
echo "deb-src http://packages.hedgerows.org.uk/raspbian wheezy/" | sudo tee -a
/etc/apt/sources.list.d/packages.hedgerows.org.uk.list

  2. Install the repository signing key: 
    sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 0F5BFDFE

  3. Update your package index: 
    sudo apt-get update

Removing this repository

  1. Remove the repository: 
    sudo rm /etc/apt/sources.list.d/packages.hedgerows.org.uk.list

  2. Remove the repository signing key: 
    sudo apt-key del 0F5BFDFE

  3. Update your package index: 
    sudo apt-get update
